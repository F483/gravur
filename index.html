Secure

No data sent to third parties! Everything is stored on the bitcoin blockchain and all locally cached data is encrypted.

Complete API

All features can be integrated with your software via JSON-RPC protocol, Python interface or command line interface.

Identities

Create signed aliases for a bitcoin addresses, thus enabling its usage with apps.

Messaging

Censorship resistant messaging, with public broadcasts (like tweets) and private encrypted messages.

Signing

Cryptographically sign documents, which is also an implicit proof of existence.

Invoices

Easily send descriptive payment request to your clients.
